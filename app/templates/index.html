Introduction

This project focuses on predicting ATP binding sites within protein sequences.

Approach

Feature extraction is performed using Position-Specific Scoring Matrix (PSSM), binary encoding, and sliding window techniques. A combination of machine learning models including Random Forest and XGBoost. We also used deep learning model such as ESM, MAMBA, and MSCNN are employed to enhance prediction accuracy. The results are visualized and presented through a web-based interface for ease of access and interpretation.

Architecture Overview

The website predicting tool combines sequence analysis and machine learning methods to predict ATP binding sites in transport protein sequences. The flow chart includes different steps of traditional machine learning, deep learning, and PLM.
