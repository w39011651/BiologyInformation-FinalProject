## 使用說明

本專案未包含 LC-PLM 預訓練模型與原始碼，Docker build 時將自動從 [amazon-science/LC-PLM](https://github.com/amazon-science/LC-PLM) 下載最新版本及權重。如需手動執行，請預先下載並放置於 `/LC-PLM` 目錄。
